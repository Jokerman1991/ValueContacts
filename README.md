# ValueContacts

一个简易的T9拼音智能拨号软件

●开发背景：魅族Flyme系统拨号早期没有T9拼音智能拨号，用户体验很差，索性开发了一个。<br>
●软件现状：当时软件命名为“魅拨号”，在魅族应用商店发布以后获得很大的下载量，有段时间是个人开发者下载量前十，并且获得了很多的用户好评。自从Flyme更新了T9拨号以后，“魅拨号”的使命也就完成了，被魅族应用商店的运营给下架了。<br>

●源码中你可能感兴趣的知识点<br>
⒈增删改查系统联系人以及SIM卡联系人；<br>
⒉查询和删除通话记录；<br>
⒊中文转英文全拼；<br>
⒋T9数字检索联系人；<br>
⒌列表的字母导航；<br>
⒍颜色样式的统一使用；<br>
